## Purpose Of This Project
### Description 
- This project teaches us how end to end project works in real world . Churn Detection model is trained . 
- Feature Engineering and Data Cleaning Process have been applied. In addition , Data Validation has been made using Great Expectations library.
- Optuna Library has been used To find the best hyperparameter for the model
- MLWorkflow library has been used to log artifacts
